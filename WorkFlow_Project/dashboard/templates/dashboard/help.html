Getting Started FAQ

Getting Started

Follow these steps to set up your workspace correctly.

1

Create Clients

Before creating projects, add your clients.

Each project may / may not be linked to a client.

How to do it:

  1. Go to Clients
  2. Click Add Client
  3. Fill in client details (name, email, etc.)
  4. Click Save to create the client

Project Workflow (Recommended Order)

Instructions:

  1. Navigate to Projects
  2. Click Add Project
  3. Select a client from the dropdown
  4. Define project name and description
  5. Click Save Project
2

Create a Project

Projects are the core of the system. They contain all your tasks, sprints, and progress tracking.

3

Define Project Plan

Planning helps you stay on track and organized.

Tip: Break work into small, manageable tasks.

Instructions:

  1. Open a project from your projects list
  2. Go to Project Plan section
  3. Add tasks or milestones
  4. Set deadlines and priorities for each task

Instructions:

  1. Update task status regularly
  2. Mark completed tasks to track progress
  3. Monitor overall project progress from the dashboard
4

Track Progress

Keep your project status updated to maintain visibility and ensure timely delivery.

5

Monitor Stats

Stats give insight into performance and help you make data-driven decisions.

Instructions:

  1. Visit the Dashboard
  2. Review project completion rates and timelines
  3. Use insights to optimize future project planning
